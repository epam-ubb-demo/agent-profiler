Welcome to Agent Profiler! This section guides you through installation, first launch, and loading your first session.

## Prerequisites

- Node.js 20+
- pnpm 9+
- A supported AI coding-agent session log (e.g. GitHub Copilot CLI)

## Quick start

```bash
# Clone the repository
git clone https://github.com/epam-ubb-demo/agent-profiler.git
cd agent-profiler

# Install dependencies
pnpm install

# Start the desktop application
pnpm -F desktop dev
```
