This section documents the high-level architecture of Agent Profiler and the decisions that shaped it.

## Package structure

```
agent-profiler/
├── apps/
│   ├── desktop/      # Electron desktop application
│   └── docs/         # This documentation site
├── packages/
│   ├── core/         # Canonical types and shared logic
│   ├── ui/           # React component library
│   └── adapters-*/   # Data-source adapters
└── tooling/          # Shared build configuration
```
