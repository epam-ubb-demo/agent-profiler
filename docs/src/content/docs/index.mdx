## What is Agent Profiler?

Agent Profiler is a desktop application that ingests OpenTelemetry traces produced
by AI agent frameworks (LangChain, LlamaIndex, AutoGen, and others) and presents
them as interactive timeline views — giving you visibility into token usage, tool
calls, latency, and errors across an entire agent session.
