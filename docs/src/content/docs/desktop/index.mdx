The Agent Profiler desktop application is an Electron app that ingests OpenTelemetry traces from AI agent sessions and renders them as interactive timeline views.

## Key Features

- **Session Browser** — list, search, and open all locally available agent sessions
- **Timeline View** — flame-graph style rendering of span hierarchies, colour-coded by span type
- **Span Detail Panel** — inspect attributes, events, and resource metadata for any selected span
- **Data Source Configuration** — connect to local log directories, OTel Gateway, or Application Insights
- **Settings** — configure appearance, data paths, and adapter preferences
