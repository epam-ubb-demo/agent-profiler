## Overview

The Settings panel is accessible from the application menu or the gear icon in the toolbar. It is organised into the following sections:

## General

- **Theme** — Light, Dark, or System default
- **Language** — Interface language (English only in v1)
- **Updates** — Check for updates automatically

## Data Sources

Configure which directories and remote endpoints Agent Profiler reads sessions from. See [Data Sources](/agent-profiler/desktop/data-sources/) for a full explanation of each source type.

## Adapters

Enable or disable specific adapter parsers. Disabling an adapter prevents it from parsing sessions from that source, which can improve performance if you do not use that source type.

## Advanced

- **Log level** — controls verbosity of the application log (Error, Warning, Info, Debug)
- **Cache location** — path where parsed session data is cached
- **Reset to defaults** — restores all settings to their factory values
