This site contains the **tool-level** documentation for Agent Profiler — integration
guides, configuration references, and desktop app documentation aimed at users and
integrators.

For architecture decisions, operations runbooks, and contributor guides, visit the
[Project Documentation](https://epam-ubb-demo.github.io/agent-profiler/project/).
