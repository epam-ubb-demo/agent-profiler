## Overview

The Session Browser is the main landing view when Agent Profiler opens. It lists all agent sessions discovered from your configured data sources and lets you search, filter, and open them.

## Planned Features

- **Session list** — chronological view of all sessions with duration, model, and tool-call count
- **Search** — full-text search across session metadata and span names
- **Filters** — filter by date range, model, source adapter, or status
- **Quick preview** — hover over a session to see a summary timeline without opening it
- **Comparison mode** — select two sessions to open them side by side

## Opening a Session

Click any session in the list to open its full timeline view. Use the breadcrumb at the top of the timeline to return to the session browser.
