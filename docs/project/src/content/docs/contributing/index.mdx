## Quick Start

```bash
# Clone the repository
git clone https://github.com/epam-ubb-demo/agent-profiler.git
cd agent-profiler

# Install dependencies (requires pnpm)
pnpm install

# Build all packages
pnpm run build

# Run the desktop app in development mode
pnpm --filter @agent-profiler/desktop run dev
```

See [DEVELOPING.md](https://github.com/epam-ubb-demo/agent-profiler/blob/main/DEVELOPING.md) for the full setup guide including prerequisites and IDE configuration.
